# 🎓 Student Manager System

A simple Python project to manage student records.

## 🚀 Features
- Add student
- View students
- Delete student
- Save data in file

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling

## 📂 Project Structure
- main.py → Main program
- student.py → Student class
- database.txt → Data storage

## 🎯 Purpose
This project is built to practice real-world Python concepts and build problem-solving skills.
